# Client

To start the client :

```
yarn start
```

# Server

Pre-requisites : install [ts-node](https://github.com/TypeStrong/ts-node) and typescript globally: 

```
yarn global add ts-node
yarn global add typescript
```

To start the server :

```
ts-node -O '{"module": "commonjs"}' server.ts
```
